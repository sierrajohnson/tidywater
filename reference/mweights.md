# Molar weights of relevant compounds

A dataset containing the molar weights of several compounds in g/mol.
Column names are lowercase chemical formulas (with no charge), with the
exception of the following coagulants: alum = Al2(SO4)3*14H2O,
ferricchloride = FeCl3, ferricsulfate = Fe2(SO4)3*8.8H2O,

## Usage

``` r
mweights
```

## Format

A dataframe with one row and one column per compound
